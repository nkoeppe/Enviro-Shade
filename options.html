Env Color Banner – Rules

Rules are matched top-to-bottom against the full URL (glob). Example: *://*.pro*.cfapps.*/*

Enabled Pattern (glob) Label Color Severity Order Delete
Pattern help
  • * matches any characters; ? matches one.
  • Examples:
    • PROD: *://*.pro*.cfapps.*/*
    • QA: *://*.qa*.cfapps.*/*
    • Local HTTP: *://localhost*/*, *://127.0.0.1*/*
